# Amazon Clone

This project is a front-end clone of Amazon's homepage created using HTML and CSS.

The goal of this project was to improve my understanding of webpage layouts, responsive design, navigation bars, product sections, and styling techniques used in modern e-commerce websites.

## Features

* Amazon-inspired homepage design
* Navigation bar and search section
* Product categories and deals section
* Responsive layout

## Technologies Used

* HTML
* CSS

## My experience

By building this project, I improved my skills in CSS positioning, flexbox, responsive design, and creating professional-looking web interfaces.
